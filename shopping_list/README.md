# 🛒 Shopping List App

A simple and efficient Flutter application that allows users to manage their shopping items with real-time data updates.

---

## 📱 Features

- ➕ Add new shopping items  
- 📝 Edit existing items  
- ❌ Delete items  
- 🔄 Real-time data synchronization  
- ☁️ Backend integration using API  
- 📊 Categorized items for better organization  
- 🎨 Clean and responsive UI  

---

## 🧠 Concepts Covered

- REST API integration (HTTP requests)
- CRUD operations (Create, Read, Update, Delete)
- Asynchronous programming (Future, async/await)
- State management
- Form handling & validation
- Error handling (Snackbar, alerts)
- Dynamic list rendering
