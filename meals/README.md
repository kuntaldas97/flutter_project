# 🍽️ Meals App

A beautifully designed Flutter application that allows users to explore different meals, view detailed recipes, and mark their favorite dishes.

This project is built as part of the Udemy course  
**"Flutter & Dart - The Complete Guide"** by Maximilian Schwarzmüller.

---

## 📱 Features

- 🧾 Browse meals by categories
- 🍔 View detailed meal recipes
- ⭐ Mark/unmark meals as favorites
- 🔍 Apply filters (e.g., gluten-free, vegan, vegetarian)
- 📂 Side drawer navigation
- 📑 Tab-based navigation (Categories & Favorites)
- 🎨 Clean and responsive UI
- ⚡ Smooth navigation and animations

---

## 🧠 Concepts Covered

This project demonstrates core Flutter concepts:

- Widget tree & reusable components
- Navigation (Stack, Tabs, Drawer)
- Passing data between screens
- State management (lifting state up / Riverpod in latest versions)
- Working with lists & dynamic data
- Handling user interactions


- Theming and styling
- Animations (Hero, transitions)
